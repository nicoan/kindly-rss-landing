Kindly RSS Reader is a self-hosted RSS aggregator designed for e-ink devices
and optimized for running on low-end computers.

Features Installation Screenshots
View on GitHub

Features

  • Fetch and aggregate RSS feeds.

  • Optimized for e-ink display readability.

  • Self-hostable on low-end hardware.

  • Dark / Light Theme

Kindly RSS Reader is in its early stages of development.
You may encounter some bugs

Installation

Only Docker is available at the moment. Use the following command to run the image:

curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/nicoan/kindly-rss-reader/refs/heads/main/install_docker.sh | sh
